# Настройка GitHub Pages

## Шаги для включения GitHub Pages:

1. Перейдите в репозиторий: https://github.com/pnext5524-creator/RK-Grand
2. Нажмите на вкладку **Settings**
3. В левом меню найдите **Pages**
4. В разделе **Source** выберите **GitHub Actions**
5. Нажмите **Save**

После этого сайт будет доступен по адресу: https://pnext5524-creator.github.io/RK-Grand/
