# Настройка GitHub Pages для РК-ГРАНД

## Быстрое решение проблемы 404

1. **Перейдите в настройки репозитория**: Settings → Pages
2. **Выберите источник**: Deploy from a branch
3. **Выберите ветку**: main
4. **Выберите папку**: /docs
5. **Сохраните настройки**

## Альтернативный способ (через Actions)

1. В Settings → Pages выберите "GitHub Actions"
2. Коммитьте изменения в main ветку
3. Actions автоматически соберет и развернет сайт

## Структура проекта

- `/docs/index.html` - статическая страница для быстрого запуска
- `/.github/workflows/deploy.yml` - автоматическая сборка и деплой
- `/dist/` - папка сборки Vite (создается автоматически)
